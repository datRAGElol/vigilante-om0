Grand Theft Auto: San Andreas om0 tester

## Controls

| Key | Function |
|-----|----------|
| **Num1** | Toggle between Freeze Mode and Real-Time Mode |
| **Num2 + F** | Start an OM0 timing attempt |
| **Hold Num2** | Keeps measuring frames |
| **Release Num2** | Ends the attempt (and unfreezes the game if in Freeze Mode) |

---

## How to Use

### Freeze Mode (Default)
1. Position yourself and prepare to trigger OM0 as usual.  
2. Hold **Num2**.  
3. Press **F** to begin the frame counter.  
4. When the target frame is reached, the game will freeze immediately.  
5. Release **Num2** to unfreeze.  
6. Review the result printed on screen.

This mode is ideal for learning the exact timing window.

---

### Real-Time Mode
1. Press **Num1** to switch to Real-Time Mode.  
2. Hold **Num2**, press **F** to begin.  
3. A live counter displays the exact frame number in real time.  
4. Release **Num2** at the moment you think the offset is reached.  
5. Review the accuracy report.

This mode is ideal for real gameplay practice once you understand the freeze timing.

---

## Installation

1. Install **CLEO 5** or newer.  
2. Place the `.cs` script file into your `CLEO` directory.  
3. Launch the game.  
4. Start practicing Vigilante OM0 setups.
